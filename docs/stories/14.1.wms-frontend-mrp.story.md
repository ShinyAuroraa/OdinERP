# Story 14.1: MRP Integration — Ordens de Produção + Requisições de Material

## Status

Done

## Story

**As a** supervisor de armazém (WMS_SUPERVISOR),
**I want** visualizar as ordens de produção recebidas do MRP e o status das requisições de material,
**so that** possa monitorar a integração WMS↔MRP e acompanhar o fluxo de materiais para a produção.

## Complexity

**Estimativa:** M (Medium) · ~2-3 dias

## Dependencies

- Story 9.1 ✅, 9.2 ✅
- Backend 6.1 Done ✅ — `/api/v1/production-material-requests`

## Out of Scope

- ❌ Criação manual de ordens de produção (só via Kafka/MRP)
- ❌ Interface de configuração Kafka

## Acceptance Criteria

1. **Página `/mrp`** — requisições de material:
   - DataTable: ID, Ordem Produção, Produto, Qtd Solicitada, Status (PENDING/RESERVING/PICKING_PENDING/DELIVERED/STOCK_SHORTAGE/COMPENSATED), Data
   - Filtros: status
   - Badge de alerta para STOCK_SHORTAGE
   - staleTime 60s (dados de integração)

2. **React Query hooks** — `lib/api/mrp.ts`:
   - `useMaterialRequests(filters?)` — GET /production-material-requests, staleTime 60s
   - `useMaterialRequest(id)` — GET individual

3. **TypeScript types** — `types/mrp.ts`:
   - `MaterialRequestStatus`, `ProductionMaterialRequest`, `ProductionMaterialRequestItem`

4. **Sidebar**: rota `/mrp` → `implemented: true`

5. **Testes unitários** (≥ 6 novos)
